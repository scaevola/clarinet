fn get_tsbuildinfo(
    &self,
    specifier: &ModuleSpecifier
) -> Result<Option<String>, AnyError>

Get the optional build info from the cache for a given module specifier. Because build infos are only associated with the “root” modules, they are not expected to be cached for each module, but are “lazily” checked when a root module is identified. The emit_type also indicates what form of the module the build info is valid for.
